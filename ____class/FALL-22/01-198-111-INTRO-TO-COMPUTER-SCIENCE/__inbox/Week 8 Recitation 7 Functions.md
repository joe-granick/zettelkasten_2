## Method signatrues
`public static returnType methodName(parameterType parameterName,..)`
- starts with 
	- `public`
	- `static`
	- **return type**
		- `void`
		- `int`
		- `double`
		- `boolean`
		- `String`
		- `int[]`
		- etc
	- parameter data type and parameter variable(s)
